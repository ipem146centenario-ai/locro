Sistema

El concepto de sistema se refiere a un conjunto de elementos que están relacionados entre sí y trabajan juntos para cumplir una función o alcanzar un objetivo.

Sistema Digital

Un sistema digital es un conjunto de dispositivos electrónicos que procesan, almacenan o transmiten información usando señales digitales representadas por 0 y 1.

Características principales

  • • Utiliza señales binarias (0 y 1).
  • • La información se procesa mediante circuitos lógicos.
  • • Es más preciso y resistente al ruido que los sistemas analógicos.

Componentes básicos

  • Entradas: reciben datos (teclado, sensores).
  • Procesamiento: puertas lógicas y circuitos digitales.
  • Salidas: muestran o envían el resultado (pantalla, impresora).

Ejemplos

  • • Computadoras
  • • Teléfonos celulares
  • • Calculadoras
  • • Relojes digitales
  • • Consolas de videojuegos
✔ Un sistema digital trabaja con información en forma de bits (0 y 1).

Sistema Analógico

Un sistema analógico representa y procesa información mediante señales continuas.

Características

  • • Utiliza señales continuas.
  • • Los valores cambian de forma progresiva.
  • • Es más sensible al ruido.

Ejemplos

  • • Reloj con agujas
  • • Radio analógica
  • • Termómetro de mercurio
✔ Un sistema analógico usa señales continuas.

BIT

Un bit (binary digit) es la unidad más pequeña de información en informática.

Características

  • • Solo puede tener dos valores: 0 o 1.
  • • Representa estados como encendido/apagado.

Relación con otras unidades

  • • 8 bits = 1 byte
  • • Los bytes representan letras, números y símbolos.
✔ Un bit es la unidad básica de información digital.

Sistema Binario

El sistema binario utiliza solo dos números: 0 y 1.

Características

  • • Su base es 2.
  • • Cada posición representa una potencia de 2.
  • • Es el sistema utilizado por las computadoras.
Ejemplo: 1011 = 8 + 0 + 2 + 1 = 11

Conversiones

Decimal → Binario

Dividir entre 2 y tomar los restos.

Binario → Decimal

Sumar potencias de 2 según cada bit.

Hardware

El hardware se refiere a todas las partes físicas de una computadora... componentes tangibles que puedes ver y tocar.

Componentes de PC

1. Procesador (CPU)

Función: Ejecuta instrucciones y controla el PC.

  • • Núcleos: Más núcleos = mejor multitarea.
  • • Frecuencia (GHz): Velocidad de operación.

2. Placa base (Motherboard)

Une todos los componentes y permite la comunicación entre ellos.

3. Memoria RAM

Almacena temporalmente datos para uso rápido de la CPU.

4. Almacenamiento

  • • HDD: Económico, gran capacidad, lento.
  • • SSD: Muy rápido, carga veloz de programas.
Recomendación: Usar SSD para el sistema y HDD para archivos grandes.

5. Tarjeta gráfica (GPU)

Procesa gráficos y video. Vital para juegos y diseño.

Software: El "Cerebro" Lógico

El software es el conjunto de instrucciones, datos y programas que permiten que el hardware realice tareas específicas. Si el hardware es el cuerpo, el software es la mente.

¿Cómo funciona? El software traduce nuestras necesidades a un lenguaje que el procesador entiende (binario), pasando por diferentes niveles de abstracción.

1. Clasificación Detallada

Sistemas Operativos

Es el software base. Gestiona los recursos del hardware y permite la interacción con el usuario.

  • • Ejemplos: Windows, macOS, Linux, Android, iOS.

Software de Aplicación

Programas diseñados para realizar tareas específicas para el usuario final.

  • • Ejemplos: Office, Navegadores, Editores de fotos, Juegos.

2. Software de Programación

Son las herramientas que usan los desarrolladores para crear otros programas. Incluyen editores de texto, compiladores e intérpretes.

Lógica de programación: Entrada → Algoritmo (Proceso) → Salida

3. Licenciamiento: ¿Libre o Propietario?

Tipo Características Ejemplos
Software Libre Se puede ver el código, modificarlo y compartirlo libremente. Linux, VLC, LibreOffice.
Software Propietario El código es cerrado y pertenece a una empresa; suele requerir pago. Windows, Photoshop, iOS.

4. Firmware: El puente

Es un tipo de software específico que está grabado directamente en los circuitos integrados (hardware). Su función es controlar el funcionamiento del dispositivo a nivel físico. Ejemplo: La BIOS de una computadora.

✔ Sin software, el hardware es solo metal y plástico sin utilidad. El equilibrio entre ambos define el rendimiento de un sistema.
